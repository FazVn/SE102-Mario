# Test Cases

## Scene

- [ ] Mở game vào MenuScene.
- [ ] Từ MenuScene chuyển sang PlayScene.
- [ ] Từ PlayScene chuyển sang GameOverScene.
- [ ] Từ PlayScene chuyển sang WinScene.
- [ ] Restart không crash.

## Mario

- [ ] Mario đứng trên nền không rơi xuyên.
- [ ] Mario chạy trái/phải.
- [ ] Mario nhảy.
- [ ] Mario rơi xuống hố thì chết hoặc restart.
- [ ] Mario không kẹt vào tường.

## Enemy

- [ ] Goomba đi ngang.
- [ ] Goomba quay đầu khi gặp tường.
- [ ] Mario nhảy lên Goomba thì Goomba bị hạ.
- [ ] Mario chạm ngang Goomba thì Mario chết.

## Render

- [ ] Sprite hiển thị đúng vị trí.
- [ ] Camera theo Mario.
- [ ] Sprite không bị lệch khi camera di chuyển.
- [ ] Flip sprite hoạt động nếu có triển khai.

## Level

- [ ] Load được demo_level.txt.
- [ ] Object được tạo đúng vị trí.
- [ ] Portal hoặc vùng kết thúc màn hoạt động.
